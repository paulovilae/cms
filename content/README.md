# Content Directory

This directory contains text-based content for various platforms and purposes.

## Purpose

- Centralized location for all content separate from code implementation
- Organized structure for managing different content types
- Facilitates content reuse and consistency across the platform

## Subdirectories

- **web/** - Content for web pages, blog posts, and online materials
- **marketing/** - Marketing materials, promotional content, and campaign assets
- **documentation/** - End-user documentation, help guides, and tutorials

## Usage Guidelines

- Maintain consistent voice and tone across all content
- Follow established content style guides
- Use descriptive filenames that reflect content purpose
- Include appropriate metadata and tagging
- Coordinate with the Content Creator agent for content creation and management
- Consider SEO best practices for web-oriented content
- Regularly review and update content for accuracy
