# Reports Directory

This directory contains agent-generated reports organized by the agent type that created them.

## Purpose

- Centralized location for all agent-generated analyses and documents
- Organized structure to easily find reports by agent type
- Clear separation of responsibilities between different agent roles

## Subdirectories

- **analytics/** - Data analysis reports and insights from the Analyst agent
- **architecture/** - System design documents and technical specifications from the Architect agent
- **coordination/** - Project planning and workflow documents from the Orchestrator agent
- **documentation/** - Documentation files created by the Documentor agent
- **performance/** - Performance test results and optimization reports from the Performance Engineer
- **security/** - Security assessments and vulnerability reports from the Security Expert
- **ux/** - User experience reviews and audits from the UX Expert

## Usage Guidelines

- Each agent should only create or modify files in their designated subdirectory
- Use consistent file naming conventions for reports (e.g., `YYYY-MM-DD-report-name.md`)
- Include proper metadata and context in each report
- Cross-reference related reports when applicable
