# Completed Tasks

This directory contains tasks that have been successfully completed.

Tasks are moved here from the `active` directory once they meet all acceptance criteria and have been reviewed if needed.

## Directory Organization

Completed tasks are stored with their original filenames, preserving their creation date and agent information. This helps maintain a historical record of work completed.

## Retention Policy

Completed tasks should be retained for at least 6 months to provide historical context and reference. After that period, they may be archived if needed to manage repository size.

## Relationship to ConPort

When tasks are moved to this directory, their status should also be updated in ConPort using the `update_progress` tool to maintain consistency between the file system and the ConPort database.
