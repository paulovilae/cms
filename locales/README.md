# Locales Directory

This directory contains internationalization and localization resources for the project.

## Purpose

- Centralized location for all language translations and localization assets
- Organized structure for managing translations by language
- Support for multi-language and multi-region content

## Subdirectories

- **en/** - English language resources (base/default language)
- **es/** - Spanish language resources
- Additional language directories as needed for project requirements

## Usage Guidelines

- Maintain consistent naming conventions for translation keys
- Document context for translators to ensure accurate translations
- Address cultural considerations for each language
- Implement proper handling of pluralization and grammatical variations
- Regularly update translations when source content changes
- Test localized content with native speakers when possible
- Coordinate with the Localization Expert agent for translation management
- Consider implementing translation memory and glossaries for consistency
