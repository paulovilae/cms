# Capacita Training Platform Assets

This directory contains visual and design assets specific to the Capacita AI Training Platform.

## Purpose

- Centralized location for Capacita-specific visual elements
- Consistent design resources that maintain the Capacita brand identity
- Training and education-focused icons, images, and styles

## Subdirectories

- **icons/** - Education-specific icons, achievement badges, and UI symbols
- **images/** - Capacita platform screenshots, learning illustrations, and marketing images
- **styles/** - Capacita-specific CSS, SCSS, and theming files

## Usage Guidelines

- Assets should reflect the Capacita Training Platform branding
- Follow the educational platform design system guidelines
- Maintain proper image optimization for web usage
- Ensure training-related imagery is engaging and motivational
- Document usage context for specialized learning elements
- Coordinate with the Graphic Designer agent for asset creation
- Consider accessibility for learning-focused assets
