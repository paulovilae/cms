# General Assets

This directory contains visual and design assets that are shared across all business units of the CMS platform.

## Purpose

- Centralized location for platform-wide visual elements
- Consistent design resources that apply to all business units
- Shared styles, icons, and images for the core platform

## Subdirectories

- **icons/** - Shared icon sets, SVG icons, and other iconography
- **images/** - Shared photographs, screenshots, and raster image files
- **styles/** - Global CSS, SCSS, and other styling files

## Usage Guidelines

- Assets in this directory should be truly cross-platform
- Business-specific assets should be placed in business-specific directories
- Maintain proper image optimization for web usage
- Follow naming conventions for clarity and organization
- Document the source and usage context for each asset
