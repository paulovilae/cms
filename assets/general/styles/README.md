# General Platform Styles

This directory contains style assets that are shared across all business units of the CMS platform.

## Purpose

- Centralized location for platform-wide styling
- Consistent UI components and theming
- Core design system implementation

## Guidelines

- Follow the established CSS/SCSS architecture
- Document variables, mixins, and utility classes
- Maintain backward compatibility when updating
- Use a modular approach for better maintainability
- Keep business-specific styles in their respective directories
- Comment code thoroughly for easier maintenance

## Contents

This directory typically includes:

- Global variables (colors, typography, spacing)
- Base styles (resets, typography, utilities)
- Common UI components
- Animation libraries
- Responsive design foundations
