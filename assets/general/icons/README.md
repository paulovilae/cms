# General Platform Icons

This directory contains icon assets that are shared across all business units of the CMS platform.

## Purpose

- Centralized location for platform-wide icons
- Consistent iconography for core UI elements
- Shared navigation, action, and system icons

## Guidelines

- Icons should follow a consistent style guide
- SVG format is preferred for scalability
- Include both light and dark theme versions when applicable
- Document the intended usage for each icon
- Name files descriptively (e.g., `user-profile.svg`, `settings-gear.svg`)
