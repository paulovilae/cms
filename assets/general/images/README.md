# General Platform Images

This directory contains image assets that are shared across all business units of the CMS platform.

## Purpose

- Centralized location for platform-wide imagery
- Consistent visual elements for core UI components
- Shared backgrounds, illustrations, and photos

## Guidelines

- Images should be optimized for web usage
- Include multiple resolutions for responsive design
- Maintain a consistent visual style
- Document the intended usage for each image
- Use descriptive filenames (e.g., `platform-background.jpg`, `welcome-illustration.png`)
- Include attribution and licensing information when required
