# Salarium HR Platform Assets

This directory contains visual and design assets specific to the Salarium HR Platform.

## Purpose

- Centralized location for Salarium-specific visual elements
- Consistent design resources that maintain the Salarium brand identity
- HR-focused icons, images, and styles

## Subdirectories

- **icons/** - HR-specific icons, workflow indicators, and UI symbols
- **images/** - Salarium platform screenshots, HR illustrations, and marketing images
- **styles/** - Salarium-specific CSS, SCSS, and theming files

## Usage Guidelines

- Assets should reflect the Salarium HR Platform branding
- Follow the HR platform design system guidelines
- Maintain proper image optimization for web usage
- Ensure HR-related imagery is professional and inclusive
- Document usage context for specialized HR elements
- Coordinate with the Graphic Designer agent for asset creation
