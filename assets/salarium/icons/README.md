# Salarium HR Platform Icons

This directory contains icon assets specific to the Salarium HR Platform.

## Purpose

- Centralized location for HR-specific icons
- Consistent iconography for Salarium UI elements
- HR workflow, personnel, and process symbols

## Guidelines

- Follow the Salarium HR Platform design language
- SVG format is preferred for scalability
- Include both light and dark theme versions
- Document the intended usage for each HR icon
- Use descriptive filenames that indicate HR function (e.g., `employee-profile.svg`, `review-process.svg`)
- Include category prefixes for organization (e.g., `profile-user.svg`, `workflow-approval.svg`)

## Categories

- User and employee management
- HR workflow processes
- Document and form indicators
- Job posting and recruitment
- Performance review and feedback
- Compensation and benefits
- Time tracking and attendance
