# Latinos Trading Platform Icons

This directory contains icon assets specific to the Latinos Trading Platform.

## Purpose

- Centralized location for trading-specific icons
- Consistent iconography for Latinos UI elements
- Trading indicators, chart patterns, and specialized symbols

## Guidelines

- Follow the Latinos Trading Platform design language
- SVG format is preferred for scalability
- Include both light and dark theme versions for trading interfaces
- Document the intended usage for each trading icon
- Use descriptive filenames that indicate trading function (e.g., `candlestick-pattern.svg`, `trend-indicator.svg`)
- Include category prefixes for organization (e.g., `chart-line.svg`, `indicator-macd.svg`)

## Categories

- Chart types (candlestick, line, bar)
- Indicators (MACD, RSI, moving averages)
- Trading actions (buy, sell, limit, stop)
- Account management
- Navigation and UI controls
