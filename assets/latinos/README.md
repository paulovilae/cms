# Latinos Trading Platform Assets

This directory contains visual and design assets specific to the Latinos Trading Platform.

## Purpose

- Centralized location for Latinos-specific visual elements
- Consistent design resources that maintain the Latinos brand identity
- Trading-specific icons, images, and styles

## Subdirectories

- **icons/** - Trading-specific icons, charts, indicators, and symbols
- **images/** - Latinos platform screenshots, marketing images, and illustrations
- **styles/** - Latinos-specific CSS, SCSS, and theming files

## Usage Guidelines

- Assets should reflect the Latinos Trading Platform branding
- Follow the trading platform design system guidelines
- Maintain proper image optimization for web usage
- Document usage context for specialized trading elements
- Coordinate with the Graphic Designer agent for asset creation
