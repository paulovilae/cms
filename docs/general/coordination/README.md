# Coordination Documentation

This directory contains documentation related to coordinating work across different agents and teams.

## Contents

- [`agent-handover-rules.md`](./agent-handover-rules.md) - Defines proper workflows for task handover between different agent roles

## Purpose

The coordination documentation helps ensure:

1. **Clear Responsibilities**: Each task has a clear owner at any given time
2. **Proper Oversight**: Tasks are handed over through appropriate channels
3. **Effective Collaboration**: Agents work together efficiently without conflicts
4. **Quality Assurance**: Work is reviewed and validated by appropriate experts

## Usage

These documents should be referenced when:

- Planning multi-agent workflows
- Determining which agent should handle a specific task
- Establishing handover procedures for complex projects
- Training new agents on collaboration protocols
