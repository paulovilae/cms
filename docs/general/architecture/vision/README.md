# General CMS Vision Documents

This directory contains vision documents for general CMS features and platform-wide initiatives that apply across all business units.

## Vision Documents

Vision documents in this directory should focus on:
- Platform-wide architectural decisions
- Cross-business features and capabilities
- General CMS infrastructure and tooling
- Universal user experience principles

## Naming Convention

Files should follow the pattern: `vision-[feature]-[date].md`

Examples:
- `vision-universal-search-2025-01-07.md`
- `vision-multi-tenant-architecture-2025-01-07.md`
- `vision-ai-integration-platform-2025-01-07.md`
